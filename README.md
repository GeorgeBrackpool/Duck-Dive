# Duck Dive

**Duck Dive** is a fast-paced endless runner mobile arcade game built in Unity. Players guide a duck through increasingly challenging obstacles, aiming to survive as long as possible and crush their high scores. Developed as an independent project, Duck Dive was successfully shipped and released on the Google Play Store, serving as a foundational milestone in my game development journey.

---

## Features

*   **Endless Arcade Gameplay:** A dynamic, escalating obstacle system that challenges player reflexes and skill over time.
*   **Unlockable Customization:** A modular **Skin System** built using **ScriptableObjects**, allowing players to unlock and equip unique appearances for the Duck.
*   **Achievement System:** In-game milestones integrated with Google Play Games Achievements that track player progress and reward high-skill play.
*   **Mobile-Optimized Controls:** Simple, highly responsive touch controls tailored for casual, on-the-go gaming.
*   **Performance-Driven:** Optimized assets, code and build settings for a smooth framerate across a wide range of mobile devices.
*   **Score Tracking:** Local data persistence that synchronises with Google Play Games to track high scores and encourage replayability.

## About the Project & Key Takeaways

Duck Dive was a deep dive into the full mobile development lifecycle—taking a game from a raw prototype to a polished, published product. 

Building this project independently provided hands-on experience with:
*   **Data-Driven Design:** Learning & leveraging ScriptableObjects to create a clean, decoupled architecture for the unlockable skin feature.
*   **Game Mechanics & Flow:** Designing core game loops, scene management and fine-tuning collision systems.
*   **Mobile UX/UI:** Crafting intuitive touch controls and responsive user interfaces that adapt to various screen ratios.
*   **Game Feel:** Polishing based on player feedback and testing, implementing hitstop/screen shakes, and adding VFX/SFX to elevate the overall user experience.
*   **Production & Release:** Navigating the Google Play Console, managing builds, debugging mobile-specific bugs, optimizing performance and rolling out updates.

## What I Learned

This project significantly sharpened my software engineering skills, specifically in:
*   **Object-Oriented Programming (OOP):** Writing clean, reusable, and maintainable C# code within Unity's component-based ecosystem.
*   **Iterative Design:** Using player feedback to continuously balance gameplay difficulty, scoring mechanics, and progression.
*   **Resource Management:** Understanding memory optimization and mobile deployment workflows to keep the game's footprint small and efficient.
*   **Systems Implementation:** Implementing systems such as the Oxygen meter and how the systems interact with each other
